## What Was Built

A new `CurrentSessionBanner` SwiftUI view added to `DashboardView.swift` that:

1. **Header row:** terminal SF Symbol (green) + project name (bold caption) + session duration from `createdAt` ("seit Xh Ym")
2. **Token stats row:** three inline columns for Gesamt / Input / Output using `TokenFormatter.format()` with `.caption2.monospacedDigit()` font
3. **Activity footer:** last activity relative time using SwiftUI Date `.relative` style at `.system(size: 9) / .tertiary`

Style matches existing banners: `.padding(10)`, `.background(.ultraThinMaterial, in: RoundedRectangle(cornerRadius: 10))`.

The banner is conditionally rendered via `currentSession` (most recently active session by `lastActivityAt`). Frame height increased 460 → 500 to accommodate the extra content.
